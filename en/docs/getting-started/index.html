    Getting Started

    Feature

    • 40+ High-quality components out of the box.
    • Complete documentation and examples.
    • Theme customization capabilities that go deep into every detail.
    • Support typeScript，complete type definition file.
    • 90%+ unit test coverage.

    Preview

    Use the mobile phone to scan and download Tuya Panel RN App，Scan the QR code at the bottom of right panel by Tuya-RN App, you can view the currently open functional components of Tuya for use during development and accelerate the progress of panel development.

    Installation

    npm install tuya-panel-kit
    

    or
    

    yarn add tuya-panel-kit
